

La aplicacion fue construida con el comando 
npx create-react-app "pweb_canvas_react_app"

1.- La aplicacion es capaz de realizar dibujo libre
2.- Podemos cambiar el color del lapiz de dibujo a travez de botones situados en la parte inferior del lienzo
3.- Para establecer el Grosor del lapiz, podemos cambiarlo a travez de la barra deslizable situada en la parte inferior
4.- Con el Boton "Limpiar", podemos limpiar el lienzo completo para poder dibujar nuevamente en un liezo nuevo.
5.- Si asi lo queremos, podemos guardar el lienzo a travez del boton "Guardar", este boton guardara el liezo en la base de datos del navegador,
    e inmediatamente despues de guardarlo se visualizara el liezo guardado en la parte izquierda de la pantalla, donde estan el historial de lienzos.
6.- Ademas podemos guardar el contenido del lienzo como un archivo de imagen a travez del boton "Descargar"
7.- En la parte derecha tenemos las herramientas para agregar figuras geometricas(ciculo, linea recta y una figura rectangular).
    Para dibujar una figura debemos seleccionar una figura de nuestra preferencia y dibujarlo en la pizarra.
8.- En la de abajo de las herramientas de figuras tenemos la opcion para agregar cualquier texto que ingresemos.
    8.1 Para insertar texto en nuestro lienzo primero ingresaremos el texto en el input debajo del icono de ingresar texto.
    8.2 Posteriormente seleccionamos el icono de ingresar texto.
    8.3 Finalmente seleccionamos un lugar del liezo para agregar el texto en el acto.
9. El lado derecho sirve como area de visualizacion para los lienzos que tenemos guardados en la base de datos.
    9.1 Para cargar un lienzo primero debemos seleccionarlo
    9.2 Una vez el liezo fue seleccionado le damos click al boton de cargar
    9.3 El liezo esta cargado en nuestra area de trabajo.
    9.4 Si queremos quitar el lienzo que ingresamos, podemos hacerlo con el boton quitar.
